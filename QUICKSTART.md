# Planning TTT - Quick Start Guide

## 🎯 Quick Start (5 phút)

### 1. Clone & Install
```bash
git clone https://github.com/tranthaihoatth95-pixel/Planning-TTT.git
cd Planning-TTT
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Truy cập: http://localhost:5173

### 3. Build for Production
```bash
npm run build
```

## 🛠️ Build cho các nền tảng

### Web (Desktop & Mobile)
```bash
npm run build
# Output: dist/
```

### Desktop (Windows/Mac/Linux)
```bash
npm run build:desktop
# Output: ./out/
```

### Android APK (Oppo Find N)
```bash
npm run build:android
# Output: android/app/build/outputs/apk/
```
