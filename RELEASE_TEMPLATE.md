# GitHub Release Template

## 📋 Release Notes v1.0.0

### 🎉 Initial Release

Planning TTT is now available for Web, Desktop, and Android!

### ✨ Features

- **Web App**: Full-featured web application with responsive design
- **Desktop App**: Cross-platform desktop application (Windows, macOS, Linux)
- **Android App**: Native Android application optimized for foldable devices
- **Task Management**: Create, edit, and manage tasks
- **Priority System**: Set task priorities (High, Medium, Low)
- **Deadline Tracking**: Set and track task deadlines
- **Task Assignment**: Assign tasks to team members
- **Dashboard**: View statistics and task overview
- **Foldable Device Support**: Optimized for Oppo Find N

### 📱 Platforms

- **Web**: https://planning-ttt.vercel.app
- **Desktop**: [Download](https://github.com/tranthaihoatth95-pixel/Planning-TTT/releases)
- **Android**: [Google Play](https://play.google.com/store/apps/details?id=com.planning.ttt) (coming soon)

### 🛠️ Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Electron
- React Native
- Zustand

### 📥 Installation

```bash
git clone https://github.com/tranthaihoatth95-pixel/Planning-TTT.git
cd Planning-TTT
npm install
npm run dev
```
